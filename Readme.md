# Build Swingy

```
$ cd swingy
$ mvn clean package
```

# Play Swingy

```
$ cd target
```

The game can be launched in 2 modes as described below.

```
$ java -jar swingy-1.0.jar console
```

```
$ java -jar swingy-1.0.jar gui
```
